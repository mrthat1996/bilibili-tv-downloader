# bilibili-tv-downloader
 
## Requiment

- FFMPEG as global command

>Homebrew on macos
>```bash
>brew install ffmpeg
>```

>Chocolatey on windows
>```bash
>choco install ffmpeg
>```

- nodejs & npm
>Homebrew on macos
>```bash
>brew install node
>```

>Chocolatey on windows
>```bash
>choco install nodejs
>```

## Using your account

Create `.env` file
```dotenv
COOKIE="{your_cookie_string_here}"
```

## How to
1. install packages
```
npm install
```

2. Run app and do what it says
```
node main.js
```
